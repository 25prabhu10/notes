# Visual Studio Code (VSCode)

Visual Studio Code is a free source-code editor.

## Short Cut Keys

1. Command Pallet:

   `Ctrl + Shift + p` or `F1`

2. Settings:

   `Ctrl + ,`

3. Zen Mode

   `Ctrl + k` and `z`

4. Go to File Bread Crumb on the top:

   `Ctrl + Shift + ;`

5. Find a word and put cursor on each one of them:

   `Ctrl + Shift + l`
